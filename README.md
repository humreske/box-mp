# Install
## npm
`$ npm i --save box-mp`

## js
`import "box-mp"`

# How to use
`{type}{position}-{number}`
## type
m : margin

p : padding

## position
a : all

t : top

b : bottom

l : left

r : right

h : horizontal

v : vertical

## number
number*2 px
(1 <= number <= 4>)



# Some source code
